Git-full FTP

Git-full FTP is an application which provides you FTP deployment of your application via Git.


Example: ftp://<username>:<password>@<hostname>:<port>/ Example: /home/user/public_html Github: https://github.com/<username>/<repository>/archive/master.zip
Bitbucket: https://bitbucket.org/<username>/<repository>/get/master.zip
Note: If you want your code to be private, you can use Bitbucket. It provides free private git repositories.
